## 🚀 Projects

### 1. Internship Information & Monitoring System
> 📅 2026 &nbsp;|&nbsp; **Role:** Business Analyst / System Designer

**🔍 Problem:**
The Computer Engineering program tracked internships across 3 separate Excel files with manual exam-room assignment — causing duplicate data entry, scattered records, and no way to evaluate students during supervision.

**💡 Approach:**
- Interviewed stakeholders and mapped the end-to-end manual workflow to expose gaps
- Designed a centralized system replacing 3 spreadsheets with 2 integrated views
- Automated exam-room assignment for co-op supervision
- Translated requirements into technical specs, defined the data model, and documented REST APIs

**✅ Outcome:**
Replaced scattered spreadsheets and manual processes with one centralized system serving **hundreds of students per year**, with historical data migrated and a new supervision-evaluation step added.

`NestJS` `PostgreSQL` `REST API` `Docker`
